# 📖 Module 01: Basics

---

This is your first TypeScript lesson.

Goal: Understand what TypeScript is and learn the most basic types.

## 🤔 What Is TypeScript?

TypeScript is JavaScript with type checking.
It helps you find mistakes before running code.

## 🎯 Topics Covered

- string, number, boolean
- any and unknown
- void
- null and undefined
- type inference
- type annotation

## 🚀 Run This Lesson

```bash
npm run build
node dist/01_basics/index.js
```

## 🌟 Beginner Tip

Use unknown instead of any when possible.
unknown is safer because you must check type before use.
