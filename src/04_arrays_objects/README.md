# 📖 Module 04: Arrays Objects

---

This lesson explains arrays, tuples, and objects.

## 🎯 Topics Covered

- arrays and tuples
- object types
- readonly properties
- index signatures

## 💡 Key Concepts

- Array: list of same type values.
- Tuple: fixed size and fixed order values.
- Readonly: value cannot be changed after creation.
- Index signature: object with dynamic keys.
