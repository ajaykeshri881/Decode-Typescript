# 📖 Module 12: Decorators

---

This lesson gives a simple introduction to decorators.

## 🎯 Topics Covered

- class decorators
- method decorators
- property decorators

## 💡 Key Concepts

Decorator means adding extra behavior to class members.
You can log calls, validate values, or lock class changes.

## 🚀 Run This Lesson

```bash
npm run build
node dist/12_decorators/index.js
```

## 📝 Note

Decorators are optional.
Many beginners can skip this once and return later.
