# 📖 Module 03: Functions

---

This lesson focuses on functions.

## 🎯 Topics Covered

- function types
- optional parameters
- default parameters
- rest parameters
- function overloading

## 🌊 Easy Flow

1. Write a simple function with input and output types.
2. Add optional and default values.
3. Learn rest parameter for multiple inputs.
4. Learn overload when one function supports different input types.

## 🚀 Run This Lesson

```bash
npm run build
node dist/03_functions/index.js
```

---

## 🛠️ Small Practice

Create a function named squareNumber that takes number and returns number.
