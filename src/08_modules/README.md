# 📖 Module 08: Modules

---

This lesson explains modules in a very simple way.

## 🎯 Topics Covered

- export from one file
- import in another file
- module resolution basic idea

## 🧠 Basic Idea

- Write code in small files.
- Export what you need.
- Import where you want to use it.

TypeScript resolves relative paths like ./mathHelpers.

## 🚀 Run This Lesson

```bash
npm run build
node dist/08_modules/index.js
```

---

## 🛠️ Small Practice

Create a new file with one function and import it into index.ts.
