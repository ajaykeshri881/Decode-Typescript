# 📖 Module 09: Type Narrowing

---

This lesson shows how TypeScript understands exact types after checks.

## 🎯 Topics Covered

- typeof
- instanceof
- discriminated unions
- custom type guards

## 💡 Key Concepts

Narrowing means reducing a wide type to a specific type.
Example: string | number becomes only string after typeof check.

## 🚀 Run This Lesson

```bash
npm run build
node dist/09_type_narrowing/index.js
```
