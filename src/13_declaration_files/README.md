# 📖 Module 13: Declaration Files

---

This lesson introduces declaration files.

## 🎯 Topics Covered

- .d.ts basics
- using external types

## 💡 Key Concepts

A declaration file gives type information only.
It does not contain full runtime logic.

## 🚀 Run This Lesson

```bash
npm run build
node dist/13_declaration_files/index.js
```

---

## 🛠️ Small Practice

Declare one function in .d.ts and provide runtime implementation in index.ts.
