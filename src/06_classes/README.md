# 📖 Module 06: Classes

---

This lesson covers classes from simple to useful features.

## 🎯 Topics Covered

- classes and constructors
- access modifiers
- readonly and parameter properties
- getters and setters
- abstract classes

## 🔒 Access Modifiers

- public: can use anywhere
- private: only inside same class
- protected: inside same class and child classes

## 🚀 Run This Lesson

```bash
npm run build
node dist/06_classes/index.js
```
