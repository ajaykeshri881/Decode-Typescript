# 📖 Module 14: Tsconfig

---

This lesson explains important tsconfig options.

## 🎯 Topics Covered

- important compiler options
- strict mode

## ⚙️ Important Compiler Options

- strict
- target
- module
- rootDir
- outDir

## 🛡️ Why Strict Mode?

Strict mode catches more errors at compile time.
It helps beginners write safer code.
