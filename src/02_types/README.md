# 📖 Module 02: Types

---

This lesson teaches common TypeScript type tools.

## 🎯 Topics Covered

- union types
- intersection types
- type aliases
- literal types
- enums

## 💡 Key Concepts

- Union means OR. Example: string or number.
- Intersection means AND. Example: person data and contact data together.
- Literal type means fixed allowed values.

## 🚀 Run This Lesson

```bash
npm run build
node dist/02_types/index.js
```
