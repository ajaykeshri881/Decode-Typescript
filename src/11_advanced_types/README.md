# 📖 Module 11: Advanced Types

---

This lesson introduces advanced type features in simple steps.

## 🎯 Topics Covered

- conditional types
- mapped types
- template literal types

## 🌊 Easy Flow

1. Learn conditional type with yes or no result.
2. Learn mapped type by transforming each key.
3. Learn template literal type for fixed string patterns.

## 🚀 Run This Lesson

```bash
npm run build
node dist/11_advanced_types/index.js
```
