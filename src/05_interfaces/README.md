# 📖 Module 05: Interfaces

---

This lesson explains interfaces in a simple way.

## 🎯 Topics Covered

- interface basics
- extending interfaces
- interface vs type

## 📏 Easy Rule

- Use interface for object shapes.
- Use type for unions and special combinations.

Both are useful.

## 🚀 Run This Lesson

```bash
npm run build
node dist/05_interfaces/index.js
```
