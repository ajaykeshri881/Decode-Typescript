# 📖 Module 10: Utility Types

---

This lesson explains very useful built-in TypeScript utility types.

## 🎯 Topics Covered

- Partial
- Required
- Readonly
- Pick
- Omit
- Record

## 💡 Key Concepts

Utility types help you create new types from old types quickly.
You write less code and keep strong type safety.

## 🚀 Run This Lesson

```bash
npm run build
node dist/10_utility_types/index.js
```
